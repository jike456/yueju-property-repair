# 悦居物业报修系统 API 接口文档

## 概述

- **基础路径**：`/api`
- **响应格式**：统一 JSON，包含 `code`（1 成功，0 失败）、`msg`（提示信息）、`data`（返回数据）
- **认证方式**：除登录、注册外，其他接口需在请求头携带 `Authorization: Bearer <token>`

---

## 0. 文件上传（公共）

### 0.1 上传图片

| 项目 | 说明 |
|------|------|
| 路径 | `POST /api/files/upload` |
| 认证 | 是 |
| 描述 | 上传图片，返回可访问的 URL，用于报修图片、评价图片、维修处理图片 |

**请求**

- Content-Type：`multipart/form-data`
- 字段名：`file`
- 支持格式：jpg、jpeg、png、gif、webp
- 单文件最大：5MB

**响应示例**

```json
{
  "code": 1,
  "msg": "success",
  "data": {
    "url": "https://qczajmnsaztb.sealoshzh.site/uploads/2025/03/04/abc123.jpg"
  }
}
```

**前端用法**

- 报修页、评价页、完成维修：el-upload 选择文件后，调用此接口上传，将返回的 `data.url` 追加到 `images` / `processImages` 数组，再随表单提交。

---

### 0.2 删除已上传图片（可选）

| 项目 | 说明 |
|------|------|
| 路径 | `DELETE /api/files?url={url}` |
| 认证 | 是 |
| 描述 | 从服务器删除对应文件；仅允许删除本系统生成的 `uploads/年/月/日/32位hex.扩展名` 路径，防止路径穿越 |

**Query**

| 参数 | 说明 |
|------|------|
| url | 上传接口返回的完整 URL，或 `/uploads/2025/03/04/xxx.jpg` 形式（需编码） |

**说明**

- 用户从列表里移除图片时，若希望不留下孤儿文件，可调用本接口；不调用则仅前端数组变化，磁盘文件仍在。
- 文件已不存在时仍返回成功（幂等）。

## 1. 用户管理

### 1.1 用户注册

| 项目 | 说明 |
|------|------|
| 路径 | `POST /api/users/register` |
| 认证 | 否 |
| 描述 | 业主或维修工自助注册 |

**请求体 (JSON)**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名，唯一 |
| password | string | 是 | 密码（明文，后端加密） |
| realName | string | 是 | 真实姓名 |
| phone | string | 是 | 手机号，唯一 |
| role | int | 是 | 1-业主 2-维修工 |
| idCard | string | 否 | 身份证号 |
| buildingNo | string | 否 | 楼栋号（业主必填） |
| unitNo | string | 否 | 单元号（业主可选） |
| roomNo | string | 否 | 房间号（业主必填） |
| area | number | 否 | 房屋面积（业主可选） |
| moveInDate | string | 否 | 入住日期 yyyy-MM-dd |
| skillType | string | 否 | 技能类型（维修工） |
| workYears | int | 否 | 工作年限（维修工） |

**响应示例**

```json
{ "code": 1, "msg": "注册成功", "data": { "userId": 101 } }
```

### 1.6 删除用户（管理员）

| 项目 | 说明 |
|------|------|
| 路径 | `DELETE /api/users/{id}` |
| 认证 | 是（管理员） |
| 描述 | 物理删除用户及扩展表（owner/repairman）、相关通知；删除管理员前会将工单流程中的 `assigner_id` 置空 |

**限制**

- 不能删除当前登录账号；
- 不能删除最后一个管理员；
- 业主：存在报修工单时不可删；
- 维修工：`order_process` 中仍关联该维修工时不可删。

## 附录

### 通用响应码

| code | 含义 |
|------|------|
| 1 | 成功 |
| 0 | 失败（具体错误见 msg） |
| 401 | 未认证或 token 无效 |
| 403 | 无权限 |
| 500 | 服务器内部错误 |

| 值 | 含义 |
|------|------|
| 0 | 待派单 |
| 1 | 待处理 |
| 2 | 处理中 |
| 3 | 待确认 |
| 4 | 已完成 |
| 5 | 已取消 |
| 6 | 已拒绝 |

### 用户角色

| 值 | 含义 |
|------|------|
| 1 | 业主 |
| 2 | 维修工 |
| 3 | 管理员 |

### 消息类型

| 值 | 含义 |
|------|------|
| 1 | 系统通知 |
| 2 | 工单提醒 |
| 3 | 评价提醒 |
